# Duolingo Recreate
This is a simulation of the global educational app duolingo <br>
All copyrights are reserved for Duolingo

# Steps
1. Provides Explanation
2. Asks the user to make up their own sentence
3. Checks if the user has entered words that follow the basic English words constraints
4. In another exercise, the user is asked to answer two MCQ questions and records the score
5. Evaluates the score and the user's level
